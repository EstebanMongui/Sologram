## How to run

To run the sologram app at local follow the next commands:

1. *```git clone git@github.com:EstebanMongui/Sologram.git```*
2. *```cd sologram```*
3. *```npm i```*
4. *```npm start```*

then open your favorite browser at http://localhost:3000
